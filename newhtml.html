Имя Описание
UpperCase Алиасы: Upper, U. Переводит символы строки в верхний регистр
LowerCase Алиасы: Lower, L. Переводит символы строки в нижний регистр
Proper Алиасы: P. Переводит самый первый символ строки в верхний регистр
Header Алиасы: Hd. Переводит самый первый символ каждого слова строки в верхний регистр
Trim Алиасы: Tr. Отсекает крайние символы пробела слева и справа строки
Padl Алиасы: PL. В зависимости от параметров функции и типа операнда добавляет к строке слева заданное количество символов
Padr Алиасы: Pr. В зависимости от параметров функции и типа операнда добавляет к строке справа заданное количество символов
Substring Алиасы: Substr. выделяет подстроку из заданной строки
NumberFormat Алиасы: NumFormat, NF. Форматирует числовое значение согласно заданному шаблону и возвращает строковое представление
RowValue Алиасы: Rv. Возвращает значение ячейки ряда для заданного индекса таблицы и заданного имени колонки.
IIF Возвращает одно из двух заданных значений, в зависимости от заданного условия.
ROWNO Возвращает индекс текущего ряда для заданного алиаса таблицы или алиаса по умолчанию, если опущен параметр.
RANDOM Возвращает значение java.Math.random()
DEPTH Возвращает значение {@link tdo.RowState#getDepth() } для текущего ряда и заданного алиаса таблицы или алиаса по умолчанию, если опущен параметр.
ISDELETED Алиасы: IsDel. Возвращает значение true если текущий ряд удален для заданного алиаса таблицы или алиаса по умолчанию, если опущен параметр. Если ряд не удален возвращает false.
ROWSTATE Возвращает значение свойства editingState текущего ряда для заданного алиаса таблицы или алиаса по умолчанию, если опущен параметр.
BIGDECIMAL Алиасы: Dec. Преобразует заданное значение в тип java.math.BigDecimal.
DOUBLE Алиасы: Dou. Преобразует заданное значение в тип java.lang.Double.
FLOAT Алиасы: FL, REAL. Преобразует заданное значение в тип java.lang.Float.
LONG Алиасы: LO, BigInt. Преобразует заданное значение в тип java.lang.Long.
INTEGER Алиасы: Int. Преобразует заданное значение в тип java.lang.Integer.
SHORT Алиасы: Small, Smallint. Преобразует заданное значение в тип java.lang.Short.
BYTE Алиасы: Tiny, TyniInt. Преобразует заданное значение в тип java.lang.Byte.
TIMESTAMP Алиасы: TSM. Преобразует заданное значение в тип java.sql.Timestamp по заданному образцу.
TIMESTAMP Алиасы: TSM. Преобразует заданное значение в тип java.sql.Timestamp по заданному образцу.

ACOS Соответствует функции из пакета java.Math.acos(Double)

SIN Соответствует функции из пакета java.Math.sin(Double)
TAN Соответствует функции из пакета java.Math.tan(Double)
EXP Соответствует функции из пакета java.Math.exp(Double)
FLOOR Соответствует функции из пакета java.Math.floor(Double)
LOG Соответствует функции из пакета java.Math.log(Double)
RINT Соответствует функции из пакета java.Math.rint(Double)
ROUND Соответствует функции из пакета java.Math.round(Double)
SQRT Соответствует функции из пакета java.Math.sqrt(Double)
TODEGREES Соответствует функции из пакета java.Math.toDegrees(Double)
